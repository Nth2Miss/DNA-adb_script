# 二重螺旋自动化脚本 - 项目说明

## ✨ 项目简介
基于 Android ADB 开发的自动化脚本，专为《二重螺旋》游戏设计，实现自动化操作功能。

## 📱 设备配置
- **设备型号**：华为 MatePad Air
- **屏幕分辨率**：2800 × 1840 px
- **适配状态**：🟡 分辨率计算功能开发中

## 🚀 功能特性
- ✅ ADB 自动化操作
- 🔄 多分辨率兼容（开发中）
- ⚡ 高效稳定的自动化流程

```

## 🛠️ 技术要求
- Android Debug Bridge (ADB)
- Python 3.7+
- 已开启开发者选项的 Android 设备

## 🔧 使用说明
1. 连接 Android 设备并启用 USB 调试
2. 安装必要的 Python 依赖包
3. 配置设备参数（目前适配 MatePad Air 原生分辨率）
4. 运行主程序开始自动化操作

## ⚠️ 注意事项
- 当前版本暂未实现动态分辨率计算
- 请确保设备分辨率设置为 2800×1840
