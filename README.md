# 二重螺旋自动化脚本 - 项目说明

## ✨ 项目简介

基于 Android ADB 开发的自动化脚本，专为《二重螺旋》游戏设计，实现自动化操作功能。

## 📱 设备配置

- **基础适配型号**：华为 MatePad Air
- **基础分辨率**：2800 × 1840 px
- **适配状态**：🟢 已支持全局动态分辨率自适应

## 🚀 功能特性

- ✅ ADB 自动化操作
- ✅ 多分辨率自动兼容（基于坐标缩放算法）
- 🖥️ 可视化图形控制台 (GUI)

## 🛠️ 技术要求

- Android Debug Bridge (ADB)
- Python 3.7+
- 已开启开发者选项（USB调试/无线调试）的 Android 设备

## 🔧 使用说明

1. 连接 Android 设备并启用 USB 调试（支持无线模式）
2. 安装必要的 Python 依赖包 (如 `opencv-python`, `Pillow`, `PyQt6`, `qfluentwidgets` 等)
3. 运行 `gui_main.py` 启动可视化控制台
4. 在控制台中选择设备与对应脚本，开始自动化操作

## ⚠️ 注意事项

- **坐标转换**：脚本底层以 2800×1840 为基础分辨率进行开发，现已加入自动坐标转换机制。
- **设备兼容**：绝大多数手机/平板均可直接兼容，但比例极度特殊（如超长带鱼屏）的设备可能会有微小点击偏差。
- **连通性调试**：如果无法连通手机，可以尝试电脑 ping 手机/平板；如果无法 ping 通，则需要手机使用终端类 APP（比如 MT 管理器中的 Terminal）反向 ping 电脑 IP 以确认双向连通性。
